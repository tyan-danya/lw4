# Webpack Example for students

Пример сборки фронтенда для студентов МРМТ (4 курс). Возможности сборки:

- транспиляция и объединение js (babel)
- модульная сборка css
- минификация js/css

## Установка

Убедитесь что установлен Node.js

```bash
npm i
```

## Запуск сборки

```bash
npm run build
```
